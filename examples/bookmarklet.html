🌸 Blossom Media — Bookmarklet

Öffne die Mediathek auf jeder beliebigen Webseite.
Nach der Auswahl wird die URL automatisch in die Zwischenablage kopiert.

Wie funktionierts?

1

Ziehe den folgenden Link in die Lesezeichenleiste deines Browsers (oder Rechtsklick → Lesezeichen hinzufügen):

⬇️ Bookmarklet wird unten nach Konfiguration generiert.

2

Auf einer beliebigen Webseite das Lesezeichen klicken → die Mediathek öffnet sich als Overlay.

3

Datei hochladen oder aus der Mediathek auswählen → die URL wird in die Zwischenablage kopiert und kann überall mit Strg+V eingefügt werden.

Wie es technisch funktioniert

Das Bookmarklet lädt das Widget-Script (blossom-media.iife.js) dynamisch in die aktuelle Seite. Das Script ist vollständig in einer Shadow DOM-Kapsel isoliert — es beeinflusst keine Styles der Host-Seite und umgekehrt.

Der onInsert-Callback kopiert die resultierende URL über die Clipboard API in die Zwischenablage. Dadurch funktioniert es auf jeder Seite — ohne dass ein Input-Feld vorhanden sein muss.

Manueller Einbau (fortgeschritten)

Falls du den Bookmarklet-Code lieber selbst einbaust:

javascript:void(function(){
  if(window.__blossomMedia){
    window.__blossomMedia.open();
    return;
  }
  var s=document.createElement('script');
  s.src='YOUR_WIDGET_URL';
  s.onload=function(){
    window.__blossomMedia=window.BlossomMedia.init({
      servers:['https://blossom.primal.net'],
      relayUrl:'wss://relay.damus.io',
      targets:'',
      onInsert:function(result){
        var text=result.formattedText||result.url;
        navigator.clipboard.writeText(text).then(function(){
          var d=document.createElement('div');
          d.textContent='\u2705 Kopiert! ('+result.insertMode+')';
          d.style.cssText='position:fixed;bottom:1rem;right:1rem;'+
            'background:#6c63ff;color:#fff;padding:0.75rem 1.25rem;'+
            'border-radius:8px;font:600 0.9rem system-ui;z-index:2147483647;'+
            'box-shadow:0 2px 12px rgba(0,0,0,0.3);';
          document.body.appendChild(d);
          setTimeout(function(){d.remove()},3000);
        });
      }
    });
    window.__blossomMedia.open();
  };
  document.head.appendChild(s);
}())
